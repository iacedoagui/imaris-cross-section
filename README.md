# imaris-cross-section
A script to obtain the cross section on a 3D object inside an Imaris (.ims) file

# Libraries to Include are
numpy: Provides efficient array handling and mathematical operations.
Pillow: Converts NumPy arrays into image files (e.g., PNG) for saving and visualizing 2D slices of the 3D dataset.
scipy: Offers advanced image processing functions, such as rotation and interpolation, to manipulate and align the 3D data.
imaris-ims-file-reader: Loads .ims microscopy files into Python for further analysis and processing.
